# PDQC-Enquiry-Management
A web-based application for managing product enquiries, part numbers, dashboards, and monthly reports.

## Objectives

- Centralize enquiry management
- Replace Excel-based tracking
- Upload monthly enquiry files
- Search historical enquiries
- Dashboard and analytics
- User authentication
- AI-powered recommendations (Future)

## Tech Stack

- Python
- Streamlit
- SQLite
- Pandas
- Plotly
